# Resume — AI 求职陪跑教练

Next.js 15 MVP. Design docs are kept locally in `docs/` (not pushed).

## Prerequisites
- Node.js 24 (see `.nvmrc`)
- npm 11+

## Getting started

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Stack (MVP)
- Next.js 15 (App Router) · React 19 · TypeScript
- TailwindCSS · shadcn/ui
- Prisma · Supabase Postgres + pgvector
- Auth.js v5

## WSL2 note
If developing on WSL with the project on a Windows-mounted path, enable file-watch polling (see `.env.example`).
